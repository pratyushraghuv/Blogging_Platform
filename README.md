# 📝 Blogging Platform with Authentication & Admin Panel

## 🚀 Overview

This project is a **full-stack Blogging Platform** developed using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

It provides a secure and scalable environment where users can create, manage, and interact with blog content. The platform includes authentication, CRUD operations, and an admin panel for managing users and posts.

---

## ✨ Features

### 👤 User Features

* Secure User Registration & Login (Authentication)
* Create, Edit, and Delete Blog Posts
* View and Read Blogs
* Comment on Posts
* Responsive UI for smooth user experience

### 🛠️ Admin Features

* Admin Dashboard
* Manage Users
* Control/Delete Blog Content
* Monitor Platform Activity

---

## 🧰 Tech Stack

**Frontend**

- React.js  
- React Router DOM  
- Tailwind CSS  
- Axios  
- Shadcn/ui  
- CSS / Bootstrap  

**Backend**

- Node.js  
- Express.js  
- Multer (for file uploads)  
- Cloudinary (for media storage)  
- Mongoose (MongoDB ORM)  
- Cookie Parser  
- Bcrypt (for password hashing)  

**Database**

- MongoDB (via Mongoose)

## 📡 API Overview

| Method | Endpoint   | Description   |
| ------ | ---------- | ------------- |
| POST   | /register  | Register user |
| POST   | /login     | Login user    |
| GET    | /blogs     | Get all blogs |
| POST   | /blogs     | Create blog   |
| PUT    | /blogs/:id | Update blog   |
| DELETE | /blogs/:id | Delete blog   |
